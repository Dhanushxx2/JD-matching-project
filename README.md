# AI-Based JD Matching System

## Overview
This project uses NLP (TF-IDF + Cosine Similarity) to match job descriptions with student profiles.

## Features
- JD-based candidate matching
- Match % scoring
- Email automation
- Excel upload support
- Cloud deployment (Streamlit)

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
